# 나만의 자소서 도우미

## 프로젝트 목적

한국 취업 자소서의 6대 결함(추상 표현, 정량 근거 부재, 키워드 미스매치,
자기자랑식 서술, 일관성 결여, 공통 템플릿)을 탐지하고 개선 방향을 제안하는
CLI 기반 자소서 첨삭 도우미입니다.

STAR/PREP/CAR 구조 기반 첨삭, 스타일 전환, 블라인드 채용 위험 표현 점검,
멀티에이전트 분석·첨삭·최종본 생성 기능을 포함합니다.

## 실행 환경

- Python: 3.10 이상
- 실행 도구: uv
- 주요 패키지:
  - openai>=2.29.0
  - anthropic>=0.104.0
  - openai-agents>=0.17.3
  - python-dotenv
  - pydantic>=2

> SDK와 모델명은 작성 시점 기준입니다. 학습 당일 최신 버전을 다시 확인하세요.

## 실행 방법

```powershell
uv run python resume_helper.py
```

## 필요한 설정

`.env` 파일을 프로젝트 루트에 만들고 아래 키를 넣습니다.

## 주요 기능

| 명령어 | 기능 | 구현 시점 |
|--------|------|---------|
| `/help` | 도움말 출력 | Day 1 |
| `/style [스타일명]` | 첨삭 스타일 전환 | Day 2 |
| `/analyze` | 자소서 분석 (ResumeAnalysis 5필드) | Day 3 |
| `/blind` | 블라인드 채용 위험 표현 점검 | Day 5 |
| `/quit` | 종료 | Day 1 |

### 사용 가능한 스타일

- `간결형` : 추상 표현 제거, 핵심 중심 재작성
- `스토리형` : STAR 구조로 경험 재구성
- `직무맞춤형` : 직무 키워드 삽입
- `수치강조형` : 결과 수치 강조 (STAR 기반)
- `지원동기형` : PREP 구조로 지원동기 재구성

### 멀티에이전트 구조 (resume_agents.py)

```powershell
uv run python resume_agents.py
```

- `ResumeTriageAgent` : 요청 분류 및 라우팅
- `ResumeAnalyzeAgent` : 5필드 분석 + 6대 결함 탐지
- `자소서_첨삭_Specialist` : STAR/PREP/CAR 기반 개선 제안
- `자소서_최종본_Specialist` : 제출용 완성 문단 작성

## 예시 입출력

### 입력 예시
자소서 입력 > FastAPI를 활용해 로그인 API를 구현했습니다.
오류 로그를 분석해 재발 방지 문서를 만들었고,
팀원들과 코드 리뷰를 통해 완성도를 높였습니다.
입사 후 회사에 도움이 되는 개발자가 되겠습니다.

### /style 간결형 출력 예시
[간결한 첨삭가으로 첨삭 중 ...]
[첨삭 결과]
FastAPI로 로그인 API를 구현하고 오류 로그를 분석해
재발 방지 문서를 작성했습니다.
팀 코드 리뷰로 완성도를 높였습니다.
입사 후 안정적인 API 개발로 기여하겠습니다.

### /blind 출력 예시
자소서 입력 > /blind
점검할 자소서를 붙여넣으세요 > 저는 ○○대학교를 졸업했습니다.
블라인드 채용 위험 표현 후보:

'대학교' 발견 → 제거 또는 수정을 권장합니다.

## 비용 주의

- OpenAI API 호출 시 토큰 비용이 발생합니다.
- `max_completion_tokens`를 설정해 응답 길이를 제한합니다.
- 긴 자소서를 반복 입력하면 비용이 누적될 수 있습니다.
